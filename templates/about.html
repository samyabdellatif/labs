üìä Database Schema

Users Collection

  • username - Unique user identifier
  • password - User password (plaintext for demo purposes)
  • role - User role (admin/user)

Lectures Collection

  • course - Course name or code
  • instructor - Instructor name
  • days - Days of week (compressed string: '123' = SUN,MON,TUE)
  • starttime - Lecture start time (HH:MM format)
  • endtime - Lecture end time (HH:MM format)
  • numberOfStudents - Expected student count
  • lab - Laboratory number (1-6)

üîç API Endpoints
